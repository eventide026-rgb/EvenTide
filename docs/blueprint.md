# **App Name**: EvenTide

## Core Features:

- User Role Management: During sign-up, users select from roles (Owner, Planner, Guest, Security, Cohost). Access and dashboard features adapt to the user's role.
- Dynamic Event Creation: Owners create events with a simple form. The system automatically generates a unique event code and adds the event to a dynamic calendar.
- AI Invitation Design: Eni, the AI tool, uses event details and chosen colors to dynamically generate invitation card designs that reflect the event theme.
- Secure GatePass System: The system generates two QR codes: one for the event and one unique to the guest, which are included in a digital gatepass.  Security staff use their smartphones to validate the passes and admit the guest to the event.
- Event Chat Rooms: Event planners, owners and guests may engage in an event-based chat; chat section allows direct chats between participants and group chats.
- Media Sharing: Guests can upload photos/videos from their perspective.  Uploaded media forms a reel that is shared with owners, co-hosts, and planners, to enrich the event experience. 
- Real-time Announcements: A dedicated chat view allows for push notifications and instant broadcasting announcements from event organizers or administrators.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) for a sense of regality and sophistication, aligning with the event management theme.
- Background color: Light blue (#D3E2FC), a desaturated variant of royal blue to maintain a cohesive but not overwhelming presence.
- Accent color: Gold (#D4AF37), providing contrast and highlighting important interactive elements, in line with the requested color scheme.
- Body text and headline font: 'Inter' sans-serif font, for a modern and neutral feel.
- Elegant, custom icons that are intuitive and related to event planning and management.
- Clean, modern layout with clear sections for easy navigation.  The landing page will feature an event hall image, while interior pages are content-focused.
- Subtle transitions and animations to guide the user and add polish; for example, cards flipping on the guest dashboard, elements popping out of an envelope.
